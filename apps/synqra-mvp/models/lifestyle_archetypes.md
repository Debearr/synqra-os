## USAGE GUIDELINES

### Selecting an Archetype

1. **Match to Industry**: Choose based on brand vertical
2. **Consider Audience**: Select archetype that resonates with target demo
3. **Maintain Consistency**: Use same archetype across campaign
4. **Adapt Subtly**: Adjust details while keeping core energy

### Combining Archetypes

Some campaigns benefit from multiple archetypes:
- Lifestyle Couple + Global Traveler (luxury travel)
- Modern CEO + Quiet Luxury Professional (executive automotive)
- Street-Lux Creator + Urban Stylist (fashion launches)

### Custom Archetypes

When creating new archetypes, follow this structure:
- Clear name and tags
- Defined look and clothing
- Specific energy/vibe
- Shot style guidelines
- Industry fit
- Color palette
