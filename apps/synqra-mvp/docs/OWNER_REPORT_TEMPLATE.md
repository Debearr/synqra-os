# Synqra Owner Report Standard

Max Length: 150–300 words.

## Structure

### 1. WHAT CHANGED
- Bullet points. One sentence each.

### 2. WHY IT MATTERS
- Tie to business or user impact.

### 3. HOW IT WORKS (HIGH LEVEL)
- No jargon.
- Explain like to a non-technical founder.

### 4. VALIDATION QUESTIONS
- Max 3–5.
- Yes/No or small choices only.

## Forbidden
- Dense paragraphs.
- Code dumps.
- Repeating prior block context.
- Over-explaining.
